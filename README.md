# Sospeso Aurora

A transparent donation platform homepage to help refugees build food and water security with low-tech solutions.

## Features

- Story-first hero introducing a real refugee farmer
- Transparent stats bar (98% of donations reach the ground)
- 3-step process explainer
- Interactive Kenya map with project locations
- "Kick start a refugee business" tiered funding cards
- Full-bleed testimonial section

## Stack

Vanilla HTML + CSS, no build step. Open `index.html` in any browser.

## Fonts

Uses Google Fonts (Fraunces + Inter) — loaded from CDN.
